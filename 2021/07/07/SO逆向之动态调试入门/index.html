进入init函数，F5反编译

进入detect_frida_loop，可以尝试不要检测任意地址0.0.0.0,或者strcmp(&v2, "REJECT")不相等即可

双击0.0.0.0，进入只读段的汇编代码

保存

hook方法sub_B90对输入内容加密和sub_D90是一个base64编码

在sub_B90(dest, v7, "areyousure??????");获得加密拿到新dest后，v8 = strlen(dest);是dest长度， v9 = sub_D90(dest, v8);是base64加密，后续在while ( v9[v16] == v19[v16] )中判断是否校验成功，和上面if ( (*a1)->GetStringUTFLength(a1, a3) != 30 )一样判断一旦没有30位直接校验失败，只有返回1才校验成功，即v9[v16] == v19[v16]。获取此时v19的地址0xB30，base64(密文)=v19
